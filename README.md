## Task 2
Selenium script:
- Opens login page

- Closes browser

## Task 3
Calendar page:
http://da.adlynk.in:8000/html-page

## Setup Instructions

1. Clone repository
2. Run:
   composer install
   npm install
3. Run:
   php artisan serve --host=da.adlynk.in --port=8000

4. Open:
   http://da.adlynk.in:8000/login
   http://da.adlynk.in:8000/html-page
